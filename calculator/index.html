Инструкция:

  • Для удаления последнего значения необходимо нажать кнопку CE. Для очищения всей памяти — AC;
  • Для того, чтобы начать новую операцию сначала необходимо полностью очистить память (нажать CE, а затем AC);
  • Унарный минус (кнопка +/-) необходимо нажимать после ввода числа;
  • Для вычисления квадратного корня следует ввести число, затем √ и обязательно кнопку =.
